# vton3d-bat
Main Repo for Virtual Try-On Pipeline

```
Installation:
1. Create a Venv environment
2. Install Pytorch Cuda with your desired version from https://pytorch.org/get-started/locally/ 
3. Run pip install -e .
4. Create a folder `data/videos` and put your input video/s there.
5. run python scripts/extract_frames.py -n `numberofframes` (for further params see extract_frames.py)
```
