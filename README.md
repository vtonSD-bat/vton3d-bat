# vton3d-bat
Main Repo for Virtual Try-On Pipeline

```
Installation:
1. Create a Venv environment
2. Install Pytorch Cuda with your desired version from https://pytorch.org/get-started/locally/
3. Clone VGGT repo from https://github.com/vtonSD-bat/vggt
4. Change back into 3dvton-bat repo and run pip install -e . path/to/vggt
5. Run pip install -e .

```
